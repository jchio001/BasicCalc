Purpose: Does basic CS-related computations for the user.

Current functionality:
-Conversions between hec, dec, and binary numbering systems
-Generates AND, OR, and Status bitmasks
-Rejects unreasonably large inputs
-User able to enable/disbale certain functionality in the settings menu, and these settings are saved
	Current Settings:
	-Pad Binary Output (Binary display will always have a multiple of 4 digits)
	-Have input displayed as output (self-explanatory)
